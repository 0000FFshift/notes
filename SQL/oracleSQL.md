# Oracle Basics


### Operators

| Operator | Description |
| --- | --- |
| `!=`, `<>` | Not Equal |
| `=` | Equal |
| `<` | Less Than |
| `>` | Greater Than |
| `<=` | Less Than or Equal To |
| `>=` | Greater Than or Equal To |
| `IN` | Equal |
| `NOT IN` | Equal |
| `BETWEEN` | Equal |
| `NOT BETWEEN` | Equal |
